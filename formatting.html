HTML contains several elements for defining text with a special meaning.

The HTML element defines a section that is quoted from another source. Browsers usually indent <blockquote> elements.

The HTML q tag defines a short quotation. Browsers normally insert quotation marks around the quotation.

The WHO, this HTML <abbr> tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM". Marking abbreviations can give useful information to browsers, translation systems and search-engines.

The HTML <address> tag defines the contact information for the author/owner of a document or an article. The contact information can be an email address, URL, physical address, phone number, social media handle, etc.

The HTML <cite> tag defines the title of a creative work (e.g. a book, a poem, a song, a movie, a painting, a sculpture, etc.).
